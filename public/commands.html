Multi Bot Commands

Browse all available commands for Multi Bot. Use these to manage, moderate, and have fun in your Discord server!

!ban <user> [reason]
Ban a user from the server. Only available to users with ban permissions.
!kick <user> [reason]
Kick a user from the server. Only available to users with kick permissions.
!mute <user> [duration]
Mute a user for a specified duration. Requires appropriate permissions.
!unmute <user>
Unmute a user in the server.
!clear <amount>
Delete a number of messages from a channel.
!userinfo <user>
Display information about a user.
!serverinfo
Show information about the server.
!play <game>
Start a game with the bot.
!rank
Show your current level and XP.
!settings
Open the bot settings panel (if available).
!help
Show all available commands and usage information.
